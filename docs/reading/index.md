## 📖 阅读方法

我读文章一般会问自己几个问题，作为整理时的"骨架"：

1. **它解决了什么问题？** —— 不只是它做了什么，而是为什么需要做
2. **跟同类方案对比有什么优劣？** —— 真正的判断在 trade-off 里
3. **什么场景下我会用它？** —— 落到自己身上才不是空读
4. **我有什么没看懂 / 不同意？** —— 留下疑问比假装都懂更有价值
